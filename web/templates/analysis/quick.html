Como funciona
  1. Pega el texto del pliego con los requisitos.
  2. Haz clic en Evaluar Experiencia para comparar codigos UNSPSC contra la base de datos RUP de la empresa.
  3. O haz clic en Evaluar Indicadores para comparar indicadores financieros contra los datos de la empresa.
  4. Puedes lanzar ambas evaluaciones con el mismo texto o con textos distintos.
Diferencia con evaluacion completa

La evaluacion completa requiere subir el PDF y pasa por extraccion RAG (mas precisa). La evaluacion rapida usa el texto que pegas directamente, ideal para validaciones rapidas o cuando ya tienes el texto relevante.
